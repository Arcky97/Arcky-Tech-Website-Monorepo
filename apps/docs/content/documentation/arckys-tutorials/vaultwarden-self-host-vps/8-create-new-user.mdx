## Create new User
1. In the admin panel go to Invite User <ImageWithCaption src="/images/documentation/arcky-tutorials/Vaultwarden Admin Users.png" caption="Vaultwarden Admin Users" align="left" size="large"/> <ImageWithCaption src="/images/documentation/arcky-tutorials/Vaultwarden Admin Invite User.png" caption="Vaultwarden Admin Invite User" align="left" size="large"/> 
2. Enter the email for that user.
3. Go back to `https://vault.your-domain.com` and click **Create Account** <ImageWithCaption src="/images/documentation/arcky-tutorials/Vaultwarden Create Account.png" caption="Vaultwarden Create Account" align="left" size="small"/>
4. Enable **Two-step login** for extra security. <ImageWithCaption src="/images/documentation/arcky-tutorials/Vaultwarden Two-step Login.png" caption="Vaultwarden Two-step Login" align="left" size="large"/>
<Callout type="attention">
  For the **Master Password**, <Gr>write this down</Gr> on a paper and <Gr>keep it safe</Gr>. 
  <br/>You <Rd>cannot</Rd> and <Rd>should not store</Rd> this password **in Vaultwarden**.
  <br/>Because <Rd>your Vaulwarden master password is **<Rd>NOT</Rd>** recoverable</Rd>. 
  <br/>Not by you, not by the server, not even by Bitwarden devs.
  <br/>If you lose it, your vault is permanently encrypted garbage.
</Callout>
