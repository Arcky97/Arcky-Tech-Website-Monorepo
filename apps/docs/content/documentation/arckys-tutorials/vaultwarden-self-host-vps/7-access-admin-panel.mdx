## Access the admin panel
Open in your browser <br/>
`https://vault.your-domain.com/admin`
<ImageWithCaption src="/images/documentation/arcky-tutorials/Vaultwarden Admin Token.png" caption="Vaultwarden Admin Token" align="left" size="large"/> 
- Vaultwarden will ask for the **ADMIN_TOKEN** you generated earlier with:
```bash 
openssl rand -base64 48
```
- paste that token and you'll be logged into the admin interface
