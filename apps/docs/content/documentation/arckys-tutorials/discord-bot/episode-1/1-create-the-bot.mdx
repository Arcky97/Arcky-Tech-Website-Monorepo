## Create the Bot

Go to: [Discord Developers Portal](https://discord.com/developers/applications) and login with your discord account
<br/>
- Click "New Application"
<MediaGallery
  items={[
    { src: "documentation/arcky-tutorials/discord-bot-tutorial/New Application.png", width: 942, height: 227 }
  ]} /> 
- Give it a name e.g.: `MyBot`
<MediaGallery
  items={[
    { src: "documentation/arcky-tutorials/discord-bot-tutorial/Create New App.png", alt: "Create New App", width: 592, height: 460 }
  ]} /> 

- In the side menu click on "Bot" and scroll down to Intents section and enable all 3.
<MediaGallery
  items={[
    { src: "documentation/arcky-tutorials/discord-bot-tutorial/App Intents.png", alt: "App Intents", width: 931, height: 498 }
  ]} /> 

> Intents control what events Discord allows your bot to receive.
