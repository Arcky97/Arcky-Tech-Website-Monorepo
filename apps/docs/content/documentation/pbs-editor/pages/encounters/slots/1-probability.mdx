## Probability

<InputNumber label="Probability" value="25" width={9}/>

The <Or>Probability</Or> defines the<Gr>relative chance</Gr> of and **Encounter Slot** being selected compared to other slots within the same **Encounter Type**. 
<br/>
The total of all <Or>Probabilities</Or> for a given **Encounter Type** <Gr>does not need to equal 100</Gr>. However, if the total does equal 100, each slot's value can be interpreted directly as a <Gr>percentage chance</Gr>.
<br/>
To help with balancing, the **"Adjust Probability"** button (located below the **Encounter Slot Table**) can automatically scale all <Or>Probabilities</Or> so their total equals 100.
<br/>
This step is entirely optional—the editor will <Gr>not prevent sving</Gr> if the total differs from 100.
