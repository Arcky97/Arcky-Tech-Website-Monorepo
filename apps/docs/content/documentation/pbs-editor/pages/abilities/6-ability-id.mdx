## ID

<InputText label="ID:" value="STENCH" width={25}/>

The <Or>ID</Or> serves as the <Gr>internal identifier</Gr> used by the engine to reference an **Ability**. Each **Ability** must have a <Rd>unique ID</Rd> to function correctly.
<br/>
In most cases, the <Or>ID</Or> mirrors the **Ability's name**, formatted in <Yl>ALL CAPS</Yl> and without spaces or special characters (e.g. OVERGROW, LEVITATE).
To help prevent errors, the editor will automatically <Gr>filter out invalid characters</Gr> as you type.
<br/>
Within the PBS file, the <Or>ID</Or> defines the <Gr>start of a new Ability entry</Gr> and must always appear first in each block. This structure is handled automatically by the editor when saving, so no manual formatting is required.
