## Wiki

On the <Yl>top-right corner</Yl> of the **Ability Editor**, you'll find the Wiki button. This provides quick access to the official Essentials documentation for **Ability definitions**.
<br/>
Clicking this button will open the following page in your browser:
[Defining an Ability](https://essentialsengine.miraheze.org/wiki/Defining_an_Ability) 
<br/>
This resource contains <Gr>detailed technical documentation</Gr> for every parameter available in an **Ability PBS entry**, including engine-level behavior and supported values. It's especially useful when you need deeper insight beyond what's shown in the editor.
