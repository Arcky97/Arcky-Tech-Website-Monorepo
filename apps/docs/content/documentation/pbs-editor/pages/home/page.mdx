## Overview 

The home page gives an overview of the **latest changes** made to any entries of any PBS files **inside the editor**. 
<br/>
This has as purpose to give you an idea of what **changes were made previously** and to eventually revert these changes if wished. 
<br/>Only the **latest 10 changes** are shown here but all **older changes** can be found in the **pbs-change-log.txt** file located in `C:\Users\(user)\AppData\Roaming\Arcky's PBS Editor`.
