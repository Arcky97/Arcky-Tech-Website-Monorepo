## Berry Plant Overwrite

<InputToggle label="Berry Plant Overwrite"/>
This toggle determines how the editor handles saving the current **Berry Plant entry**:

- On — The editor will <Yl>overwrite the existing Berry Plant entry</Yl> that matches the current <Or>ID</Or> across the selected PBS file. This is the <Gr>default mode</Gr>.
- Off — The editor will save the entry as a <Gr>new Berry Plant entry</Gr>, but only if the <Or>ID</Or> does not already exist in <Yl>any loaded Berry Plant PBS file</Yl>. If a matching <Or>ID</Or> is found anywhere, the entry cannot be saved.
