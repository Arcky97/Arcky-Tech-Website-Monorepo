## Drying per Hour

<InputNumber label="dryingPerHour" value="16" width={12}/>

The <Or>DryingPerHour</Or> defines how much moisture the soil loses per hour (newer berry plant mechanics only). 
A **Berry Plant** begins with 100 moisture, and returns to 100 moisture when it is watered.
<br/>
---

### Default value 

The <Or>DryingPerHour</Or> field is <Yl>optional</Yl>, but no default value is assigned by the editor.
<br/>
If left undefined, the **Berry Plant's** <Or>DryingPerHour</Or> will be <Rd>15</Rd>
