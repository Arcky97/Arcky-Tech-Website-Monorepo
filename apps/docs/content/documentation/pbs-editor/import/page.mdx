<Header title="Importing Entries"> 
  A little guidance on how to use the Import feature
</Header>

Get the [TSV Examples](https://drive.google.com/file/d/1eT6ylkMAlfq_BmMmukd9Wf5qMXLxgGX_/view?usp=sharing) here.
Importing is not available for the Encounter, Town Map and Trainer pages due to extra complexity and it wouldn't make it easier with importing.

To create TSV files, simply use Google Sheets as Excel doesn't provide an option to save your file as TSV (Tab Separated Values). If you're file is not a TSV, you'll not be able to import it.

Each page checks your TSV file on required fields but most of these required fields are those fields that shouldn't be empty anyways so in most cases you shouldn't encounter any issues with it.
