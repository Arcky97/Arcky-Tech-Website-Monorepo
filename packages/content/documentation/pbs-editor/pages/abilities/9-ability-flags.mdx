The <Or>Flags</Or> field is a <Gr>multi-select dropdown</Gr>, allowing you to assign multiple values to an Ability at the same time.
<br/>
All selected <Or>Flags</Or> are stored in the PBS file as a <Yl>comma-separated list</Yl>, ensuring compatibility with the underlying data format.
<br/>
The available options in this dropdown can be managed through the <Gr>Ability Editor Settings</Gr>, located in the general Settings menu of the PBS Editor.
<br/>
---

### Default Value

<Or>Flags</Or> do not have a default value.
<br/>
---

### Options

By default, the editor includes the following <Or>Flag</Or>:
- FasterEggHatching
<br/>
Additional <Or>Flags</Or> can be <Gr>added or removed</Gr> via the <Yl>Ability Flags</Yl> section in the Settings menu, allowing full customization of available <Or>Flag</Or> options.
