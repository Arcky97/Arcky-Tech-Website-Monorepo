## Name

<InputText label="Name:" value="Stench" width={25}/>

The <Or>Name</Or> defines the <Gr>display name</Gr> of the **Ability** as shown to the player in-game.
<br/>
This is the human-readable label used in menus, summaries, and battle interfaces, making it an important part of how the **Ability** is presented to the player.
<br/>
---

### Default Value

The <Or>Name</Or> field is <Yl>optional</Yl>, but no default value is assigned by the editor.
<br/>
If left undefined, the **Ability's** <Or>Name</Or> will be <Rd>Unnamed</Rd>
