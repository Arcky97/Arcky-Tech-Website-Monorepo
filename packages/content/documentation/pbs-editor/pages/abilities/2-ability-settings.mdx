## Settings

Located next to the Wiki button in the <Yl>top-right corner</Yl> of the **Ability Editor**, the Settings button provides quick access to the **PBS Editor configuration**.
<br/>
When opened, it displays a filtered view of the settings panel, showing only the <Gr>General Settings</Gr> and the <Gr>Ability Editor Settings</Gr> relevant to this page.
<br/>
The **Ability Editor** currently includes the following option:

- **Ability Flags** — Allows you to <Gr>add or remove custom ability flags</Gr>, enabling the **editor** to properly recognize and handle them within **Ability entries**.
