## Hours per Stage

<InputNumber label="hoursPerStage" value="15" width={12}/>

The <Or>HoursPerStage</Or> defines the growth rate, i.e. the amount of hours between growth stages.
<br/>
---

### Default value 

The <Or>HoursPerStage</Or> field is <Yl>optional</Yl>, but no default value is assigned by the editor.
<br/>
If left undefined, the **Berry Plant's** <Or>HoursPerStage</Or> will be <Rd>3</Rd>
