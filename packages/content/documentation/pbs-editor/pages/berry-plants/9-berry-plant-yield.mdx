## Yield

<div className='flex space-x-4'>
  <InputNumber label="Yield:" value={["2", "5"]} width={10}/>
</div>

The <Or>Yield</Or> defines a range of two numbers.
The number of berries yielded by the **Berry Plant** falls within this range, and is higher the more it was watered.
<br/>
---

### Default value 

The <Or>Yield</Or> is <Yl>optional</Yl>, but no default value is assigned by the editor.
<br/>
If left undefined, the **Berry Plant's** <Or>Yield</Or> will be <Rd>2,5</Rd>
