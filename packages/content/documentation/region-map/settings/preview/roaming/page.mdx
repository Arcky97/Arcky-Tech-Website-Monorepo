  Settings for the Roaming Icons Feature.
</Header>

## Function of the Setting 

The **Roaming Icons Settings Section** includes a single Setting.

<SetVer
  v201="SHOW_ROAMING_ICONS true"
  v211="ShowRoamingIcons = true"
/>

---

## Possible Values

- `true` or `0` = Roaming Icons **<Gr>will be shown</Gr>**.
- `false` or `-1` = Roaming Icons **<Rd>will not be shown</Rd>**.
- `Switch ID` = Roaming Icons **<Or>will only be shown</Or>** when the Game Switch is `ON`.

<Callout type="hint">
  There must be at least 1 Active Roaming Species before the Roaming Mode is available.
</Callout>
