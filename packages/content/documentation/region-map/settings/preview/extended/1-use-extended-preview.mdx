## UseExtendedPreview

Use this Setting to enable or disable the Extended Location Preview Feature on the Region Map. This does require no additional setup but [UseLocationPreview](location#uselocationpreview) must be enabled. You can check [Extended Preview Setup](../../basic-setup/extended-location-preview) for more information about the different pages this Feature provides.

<SetVer
  v201="USE_EXTENDED_PREVIEW = true"
  v211="UseExtendedPreview = true"
/>

### Possible Values

- `true` = The Extended Preview **<Gr>can be used</Gr>**.
- `false` = The Extended Preview **<Rd>cannot be used</Rd>**.
