## Main Toggle

Use this Setting to enable or disable the entire Progress Counter feature.

<SetVer
  v211="ProgressCounter = true" 
  v201="PROGRESS_COUNTER = true"
/>

### Possible Values

- `true` = All Progress Counter Settings are **<Gr>used</Gr>**.
- `false` = All Progress Counter Settings are **<Rd>not used and disabled</Rd>**.
